## 📋 Descripción

Curso de **Matemáticas para niños de 1° y 2° de Primaria** diseñado con el apoyo de **LearnLM**, el modelo de IA educativa de Google. Presenta los conceptos matemáticos fundamentales de forma visual, interactiva y divertida, usando ilustraciones SVG animadas y ejemplos del mundo cotidiano.

> *"Un viaje divertido por el mundo de los números y las formas."*

Este material forma parte del proyecto **Teach LaOz Learning Management System**.

## 🎯 Objetivos del Curso

1. **Reconocer y usar los números del 0 al 1000** con confianza
2. **Aplicar conceptos espaciales** (arriba/abajo, dentro/fuera, antes/después)
3. **Dominar sumas y restas** con y sin llevadas hasta el 100
4. **Identificar figuras geométricas** básicas y sus propiedades
5. **Resolver problemas sencillos** del mundo real

## 📚 Módulos del Curso

| Módulo | Título | Temas |
|--------|--------|-------|
| 0 | Antes de los Números | Conceptos espaciales, clasificación |
| 1 | El Mundo de los Números (0-100) | Conteo, valor posicional, comparación |
| 2 | Operaciones Mágicas | Sumas y restas básicas |
| 3 | Formas y Medidas | Figuras geométricas, tiempo, longitud |
| 4 | Números Grandes (hasta 1000) | Centenas, valor posicional extendido |
| 5 | Desafíos Avanzados | Llevadas, prestadas, multiplicaciones introductorias |

## 🎨 Características del Curso

✅ **Ilustraciones SVG animadas** — personajes y objetos que cobran vida  
✅ **Ejemplos del mundo real** — galletas, manzanas, cohetes, animales  
✅ **Progresión gradual** — de lo concreto a lo abstracto  
✅ **Generado con LearnLM** — contenido optimizado pedagógicamente con IA  
✅ **Sin pantallas recargadas** — presentación limpia y amigable para niños

## 🚀 Cómo Usar Este Material

### Para Maestros

1. Revisa el [CURRICULUM.json](CURRICULUM.json) para ver la estructura completa
2. Usa las ilustraciones SVG en presentaciones o impresiones
3. Sigue el orden de módulos — cada uno prepara al siguiente
4. Adapta el ritmo según el avance del grupo

### Para Padres

1. Empieza por el **Módulo 0** para verificar que el niño tiene las bases
2. Trabaja **15-20 minutos por día** de forma consistente
3. Usa los ejemplos del mundo real (cocinar, contar objetos en casa)
4. Celebra cada logro — la confianza es clave en matemáticas
